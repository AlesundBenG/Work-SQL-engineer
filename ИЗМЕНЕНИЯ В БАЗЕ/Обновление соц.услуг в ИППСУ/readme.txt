Если услуга предоставляется сиделками, то должно подцепляться услуга с тем же наименованием, но по нулевому тарифу.

Производится это путем отлавливания услуг с одинаковыми наименованиями, которые выдает старый скрипт, и удаляем те услуги,
которые не указаны в договоре. Таким образом, если услуга разделяется на сиделку и не на сиделку, то услуг будет две, из
которых оставляем ту, которая указана в договоре.
