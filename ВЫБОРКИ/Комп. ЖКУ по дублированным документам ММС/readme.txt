Выборка людей, которым по заявлению завели документ о многодетной малообеспеченной семьи (УММС)
супругу и супруге. Из-за этого по дублированному документу может выплачиваться "Компенсация расходов на коммунальные услуги (регион.)"

Выборка только по действующим документам и назначениям на данный момент. Заявитель определяется по заявлению, по которому заведен документ
(Оригинал). Документ считается оформленным по заявлению, если дата начала действия его входит в интервал 15 дней от даты регистрации заявления.
